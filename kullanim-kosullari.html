Kullanım Koşulları

Bu web sitesini kullanarak aşağıdaki koşulları kabul etmiş olursunuz.

1. Genel Hükümler

Bu web sitesine erişen tüm ziyaretçiler, aşağıdaki kullanım şartlarını kabul etmiş sayılır. Web sitesi Çözüm Cloud tarafından yönetilmektedir.

2. Hizmet İçeriği

Çözüm Cloud, web sitesindeki bilgileri ve içerikleri dilediği zaman değiştirme hakkını saklı tutar.

3. Kullanıcı Yükümlülükleri

  • Siteyi yalnızca yasal amaçlarla kullanmak
  • Başka kullanıcıların deneyimini olumsuz etkilememek
  • Yanlış bilgi paylaşmamak

4. Sorumluluk Reddi

Web sitesindeki bilgilerin güncelliği için çaba gösterilmekle birlikte, hatalı veya eksik bilgi bulunmasından doğacak zararlardan Çözüm Cloud sorumlu tutulamaz.

5. Telif Hakları

Web sitesindeki tüm içerikler, görseller ve metinler telif hakkı ile korunmaktadır. İzinsiz kopyalanamaz.

6. Yargı Yetkisi

Doğabilecek uyuşmazlıklar Türk Hukuku kapsamında Çanakkale Mahkemeleri tarafından çözüme kavuşturulur.
